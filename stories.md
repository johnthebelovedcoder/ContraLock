1. Freelancer User Stories
Account & Trust

As a freelancer, I want to create a verified profile so that clients can trust me before inviting me to projects.

As a freelancer, I want to see my transaction history and success rate so I can track my reliability and earnings.

As a freelancer, I want verified badges on my profile so I appear more credible to potential clients.

Project Participation

As a freelancer, I want to review project terms and milestones before accepting so I know exactly what is expected.

As a freelancer, I want project terms to be locked after agreement so clients cannot change requirements mid-project.

As a freelancer, I want to see a clear milestone timeline so I can plan my work and deadlines.

Milestone Completion & Payment

As a freelancer, I want to submit deliverables per milestone so that payments are released incrementally.

As a freelancer, I want payments to be released automatically once a milestone is approved so I don’t chase clients.

As a freelancer, I want auto-approval after a set time if a client is unresponsive so my work is not unfairly delayed.

Communication

As a freelancer, I want a dedicated project chat so all communication is documented in one place.

As a freelancer, I want file sharing within chats so I don’t rely on external tools.

Disputes

As a freelancer, I want to raise a dispute on a specific milestone so issues are handled fairly.

As a freelancer, I want disputes to be resolved quickly so my income is not locked indefinitely.

As a freelancer, I want dispute decisions to be final and transparent so outcomes feel fair.

2. Client User Stories
Project Setup

As a client, I want to create a project with defined milestones so I only pay for completed work.

As a client, I want milestone templates so I don’t have to define everything from scratch.

As a client, I want to invite a freelancer via a link so onboarding is fast and frictionless.

Payment Protection

As a client, I want my funds held in escrow so I don’t risk paying without delivery.

As a client, I want to see a clear breakdown of fees and milestone amounts so pricing is transparent.

Review & Approval

As a client, I want to review submitted deliverables before payment is released so quality is ensured.

As a client, I want to request revisions on a milestone so issues can be corrected before approval.

As a client, I want to dispute a milestone if it doesn’t meet agreed criteria so my funds are protected.

Visibility & Control

As a client, I want a milestone progress tracker so I always know project status.

As a client, I want notifications for submissions and deadlines so I don’t miss approvals.

3. Dispute Arbitrator User Stories

As an arbitrator, I want to see all project agreements, deliverables, and communications so I can make an informed decision.

As an arbitrator, I want a summarized case overview so I can resolve disputes efficiently.

As an arbitrator, I want to issue binding decisions so disputes are concluded decisively.

As an arbitrator, I want to be rated after cases so quality arbitration is rewarded.

4. Platform / System User Stories
Payments & Escrow

As the platform, I want to securely hold client funds until milestones are approved so trust is maintained.

As the platform, I want to release payments automatically upon approval so the process is fast and unbiased.

Trust & Safety

As the platform, I want to verify users for high-value transactions so fraud is reduced.

As the platform, I want to track dispute patterns so repeat bad actors can be flagged or banned.

AI-Powered Enhancements

As a user, I want AI-suggested milestones so projects are structured clearly from the start.

As a user, I want AI to flag risky projects early so disputes can be prevented.

As a user, I want AI-assisted dispute summaries so resolutions are faster and clearer.
